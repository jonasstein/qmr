# qmr read and convert listmode files from MesyTec MCPD-8

Compile with 

    cmake . && make clean all runTests

Get the source
==============

go to your project directory

    cd ~/my/prj

First time: If you do not have the source yet, run  

    git clone


update the source code
======================

    cd ~/my/prj
    cd qmr
    git pull

Now compile again
